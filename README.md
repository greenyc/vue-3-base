# Ciaran Standard App

Basic app which can be used to create a Vue 3 app.

## What tech is in this app?

* Typescript
* Pinia
* Vite
* Vitest
* Vue-router
* Axios
* SCSS
* Husky

## Commands

Basic commands

### To run

```
npm i
npm run dev
```

### To build

```
npm run build
```

### To test (no coverage)

```
npm run test
```

### To test (with coverage)

```
npm run coverage
```

### To preview prod build locally

```
npm run preview
```
